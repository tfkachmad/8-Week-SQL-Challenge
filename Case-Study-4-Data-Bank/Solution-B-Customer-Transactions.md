1. What is the unique count and total amount for each transaction type?

    - Count the distinct value of each `customer_id` to find the unique counts and sum the `txn_amount` to find the total.
    - Group the final result by the `txn_type`.

    Code:

    ```sql
    SELECT txn_type
        ,COUNT(DISTINCT customer_id) AS [count]
        ,SUM(txn_amount) AS total
    FROM data_bank.customer_transactions
    GROUP BY txn_type;
    ```

    Output:

    | txn_type   | count | total   |
    | ---------- | ----- | ------- |
    | withdrawal | 439   | 793003  |
    | deposit    | 500   | 1359168 |
    | purchase   | 448   | 806537  |

2. What is the average total historical deposit counts and amounts for all customers?

    - Count all rows of each `customer_id` to find the  counts and sum the `txn_amount` to find the total deposit of each customer. Filter the result by the `txn_type` equal `deposit` and group the result by each `customer_id`.

    Code:

    ```sql
    SELECT customer_id
        ,COUNT(*) AS [count]
        ,SUM(txn_amount) AS total
    FROM data_bank.customer_transactions
    WHERE txn_type LIKE 'deposit'
    GROUP BY customer_id
    ORDER BY customer_id;
    ```

3. For each month - how many Data Bank customers make more than 1 deposit and either 1 purchase or 1 withdrawal in a single month?

   - First, find the month data from `txn_date` from each customer transactions.
   - Next, count the number of `deposit`, `purchase` and `withdrawal` using `COUNT(CASE WHEN)` to find each transaction type.
   - Count all rows and filter the result according to what the question asked, which is **customer with 1 deposit and either 1 purchase or 1 withdrawal in a single month**. Group the result by the month that already got earlier.

    Code:

    ```sql
    WITH cte
    AS (
        SELECT DATEPART(MONTH, txn_date) AS m_num
            ,DATENAME(MONTH, txn_date) AS m_name
            ,customer_id
            ,txn_type
        FROM data_bank.customer_transactions
        )
        ,cnt
    AS (
        SELECT customer_id
            ,m_num
            ,m_name
            ,COUNT(CASE
                    WHEN txn_type LIKE 'deposit'
                        THEN 1
                    ELSE NULL
                    END) AS deposit_cnt
            ,COUNT(CASE
                    WHEN txn_type LIKE 'purchase'
                        THEN 1
                    ELSE NULL
                    END) AS purchase_cnt
            ,COUNT(CASE
                    WHEN txn_type LIKE 'withdrawal'
                        THEN 1
                    ELSE NULL
                    END) AS withdrawal_cnt
        FROM cte
        GROUP BY customer_id
            ,m_num
            ,m_name
        )
    SELECT m_name AS [month]
        ,COUNT(*) AS [count]
    FROM cnt
    WHERE deposit_cnt > 1
        AND (
            purchase_cnt >= 1
            OR withdrawal_cnt >= 1
            )
    GROUP BY m_num
        ,m_name
    ORDER BY m_num;
    ```

    | -------- | ----- |
    | January  | 168   |
    | February | 181   |
    | March    | 192   |
    | April    | 70    |

4. What is the closing balance for each customer at the end of the month?

   - Find the end of month for each transaction customer made. Use `EOMONTH()` functions to do this.
   - Next, sum the total of each customer transactions and group the result by their `customer_id` and the `end_of_month` result before.
   - Finally, get the running total for customer transactions using `SUM()` function with `ROWS BETWEEN UNBOUNDED PRECEDING AND CURRENT ROW` argument to do this.

    Code:

    ```sql
    WITH cte
    AS (
        SELECT customer_id

                WHEN txn_type LIKE 'deposit'

                WHEN txn_type LIKE 'purchase'
                    OR txn_type LIKE 'withdrawal'
                    THEN txn_amount * - 1
                END AS amt
        FROM data_bank.customer_transactions
        )
        ,closing
    AS (
        SELECT customer_id
            ,end_of_month
            ,SUM(amt) AS monthly_change
        FROM cte
        GROUP BY customer_id
            ,end_of_month
        )

        ,end_of_month
    ```

5. What is the percentage of customers who increase their closing balance by more than 5%?

    - Use the previous question code to solve this question. Add the result into a temp table called `monthly_balance`.
    - The increase of closing balance will be calculated based on the increase of their balance on customer first and second month balance.
    - To do that, first use `ROW_NUMBER()` window function togive each rows a number. Each number will be partitioned by each customer_id and ordered by the date.
    - Next, filter only to get the first and second month by the row number that is `<= 2`.
    - Use `LEAD()` function to put the first and second month closing balance next to each other.

        | customer_id | end_of_month | initial_closing | closing_balance |
        |-------------|--------------|-----------------|-----------------|
        | 1           | 2020-01-31   | 312             | -640            |
        | 1           | 2020-03-31   | -640            | NULL            |
        | 2           | 2020-01-31   | 549             | 610             |
        | 2           | 2020-03-31   | 610             | NULL            |

    - Next, find the percentage change from the `intial_closing` and the `closing_balance`.
    - Finally, find the percentage of how many customer increase their balance by 5% by deviding the number of customer that increase their balance by 5% based on the previous calculation and the total customer Data Bank have.

    Code:

    ```sql
    WITH cte
    AS (
        SELECT customer_id
            ,EOMONTH(txn_date) AS end_of_month
            ,CASE
                WHEN txn_type LIKE 'deposit'
                    THEN txn_amount
                WHEN txn_type LIKE 'purchase'
                    OR txn_type LIKE 'withdrawal'
                    THEN txn_amount * - 1
                END AS amt
        FROM data_bank.customer_transactions
        )
        ,closing
    AS (
        SELECT customer_id
            ,end_of_month
            ,SUM(amt) AS monthly_change
        FROM cte
        GROUP BY customer_id
            ,end_of_month
        )
    SELECT customer_id
        ,end_of_month
        ,SUM(monthly_change) OVER (
            PARTITION BY customer_id ORDER BY end_of_month ROWS BETWEEN UNBOUNDED PRECEDING
                    AND CURRENT ROW
            ) AS closing_balance
    INTO #monthly_balance
    FROM closing
    ORDER BY customer_id;
    --
    -- Find the first and the second month of transaction amount for each customer
    WITH row_numbering
    AS (
        SELECT customer_id
            ,end_of_month
            ,closing_balance
            ,ROW_NUMBER() OVER (
                PARTITION BY customer_id ORDER BY end_of_month
                ) AS row_num
        FROM #monthly_balance
        )
        ,first_second_month
    AS (
        SELECT customer_id
            ,end_of_month
            ,closing_balance
        FROM row_numbering
        WHERE row_num <= 2
        )
        ,closing
    AS (
        SELECT customer_id
            ,end_of_month
            ,closing_balance AS initial_closing
            ,LEAD(closing_balance) OVER (
                PARTITION BY customer_id ORDER BY end_of_month
                ) AS closing_balance
        FROM first_second_month
        )
        ,closing_calculation
    AS (
        SELECT customer_id
            ,initial_closing
            ,closing_balance
            ,ROUND((CONVERT(FLOAT, (closing_balance - initial_closing)) / initial_closing) * 100, 2) AS change
        FROM closing
        WHERE closing_balance IS NOT NULL
            AND closing_balance >= 0
        )
    SELECT ROUND(CONVERT(FLOAT, COUNT(DISTINCT customer_id)) / (
                SELECT COUNT(DISTINCT customer_id)
                FROM data_bank.customer_transactions
                ) * 100, 2) AS customer_percentage
    FROM closing_calculation
    WHERE change > 5
    ```
